# Funkin-Raylib 

# How to compile  
 1. git clone it 
 ```
 git clone https://github.com/swordcube/Funkin-Raylib.git 
 ``` 
 
 2. install the haxelib 
 ``` 
 haxelib git raylib-hx https://github.com/foreignsasquatch/raylib-hx.git 
 ``` 
 3.go on your terminal and do  
 ```
 haxe build.hxml  
 ```
